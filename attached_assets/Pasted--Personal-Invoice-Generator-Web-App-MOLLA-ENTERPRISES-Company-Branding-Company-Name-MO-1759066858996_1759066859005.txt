### Tech Stack
- **Frontend:** React (Vite + Tailwind CSS)  
- **Backend:** Node.js + Express  
- **Database:** MongoDB  
- **PDF Generation:** html2canvas + jsPDF (frontend) and Puppeteer (backend)  
- **Authentication:** Not required (single-user personal use)  

---

### Features

#### 1. Dashboard
- Home page shows MOLLA ENTERPRISES branding  
- Navigation: Clients | Invoices | Settings  

#### 2. Clients Module
- Add, Edit, Delete, List clients  
- Save client details in MongoDB  

#### 3. Invoice Module
- Create invoice:  
  - **Auto-generated invoice number** (prefix "INV-", incremental: INV-0001, INV-0002, …)  
  - Select client  
  - Add items (Description, Qty, Rate, Tax, Discount)  
  - **Tax field optional** (if blank or 0, no tax applied)  
  - Auto-calculates subtotal, tax, discount, grand total  
  - Default currency: INR (₹), with option for USD, EUR, GBP etc.  
- Save invoice in MongoDB  
- Live preview in BlueOcean-style design  
- Download as PDF (frontend & backend options)  
- Invoice status: Draft, Sent, Paid, Overdue  

#### 4. Settings Module
- Update default currency  
- Add/Update payment options:  
  - Bank transfer details (Bank Name, Account, IFSC)  
  - UPI ID (text only, no QR)  
- Payment details show in PDF footer
